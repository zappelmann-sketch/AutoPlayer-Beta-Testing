# AutoPlayer Closed Testing Portal

Willkommen beim Closed Test fuer **AutoPlayer** — den innovativsten lokalen MP3-Player fuer Android mit 98+ Features, Android Auto Support und 26 Sprachen.

## Fuer Tester

### 1. App herunterladen
Die App ist ueber den Google Play Store erhaeltlich (Link folgt nach dem Launch).

### 2. Testplan oeffnen
Oeffne den interaktiven Testplan:
**[https://zappelmann-sketch.github.io/AutoPlayer-Beta-Testing/](https://zappelmann-sketch.github.io/AutoPlayer-Beta-Testing/)**

### 3. App testen
- Installiere die APK auf deinem Android-Geraet (Android 10+)
- Oeffne den Testplan im Browser
- Trage deine Geraete-Infos ein
- Teste Feature fuer Feature
- Markiere Bugs und Verbesserungswuensche

### 4. Ergebnis absenden
Am Ende des Testplans:
- Klicke **"Per E-Mail an Entwickler senden"** — die JSON-Datei wird heruntergeladen und dein Mail-Client oeffnet sich
- **ODER:** Erstelle ein [GitHub Issue](https://github.com/zappelmann-sketch/AutoPlayer-Beta-Testing/issues/new) und haenge die JSON-Datei an

## Features (Auszug)

- 98+ Features, 26 Sprachen
- 6 Player-Skins (DJ Turntable, Jukebox, Walkman, Bandmaschine, Hologramm, Standard)
- Smart DJ mit BPM/Key/Energy Analyse
- NLP Smart Playlists ("Spiel froehliche Rock Songs")
- Local Party Mode (Gaeste steuern Musik per QR-Code)
- A-B Repeat / Loop mit Practice Mode
- Waveform-Anzeige mit Lesezeichen
- Gestensteuerung (9 Gesten)
- Theme Builder (eigene Themes erstellen)
- Gehoerschutz / Dezibel-Monitor (WHO-Standard)
- Music Wrapped (wie Spotify Wrapped, aber lokal)
- Vollstaendige Sprachsteuerung + TTS Vorlesen
- Android Auto mit erweitertem Browse-Tree
- Und vieles mehr...

## Anforderungen

- Android 10 (API 29) oder hoeher
- Lokale Musikdateien (MP3, FLAC, AAC, OGG, WAV, etc.)
- Fuer Android Auto: Kompatibles Fahrzeug oder DHU
